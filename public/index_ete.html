Semestrální projekt z Client-side & Server-side

Projekt jsme umístili mimo školní server, protože aktuální nastavení tohoto serveru pravděpodobně nepodporuje single-page řešení, které jsme použili. Pro single-page řešení jsme použili technologii React. Byla snaha upravit htaccess, ale nepodařilo se najít konfiguraci, která by zprovoznila naši aplikaci. Zvolili jsme náhradní řešení v podobě externího hostingu u WebGlobe a Azure. Na WebGlobe hostujeme front-end aplikaci a v Azure provozujeme databázi Postgres a back-end na platformě .NET Core.

Front-End: kodix.cz

Zdrojový kód: GitHub

Back-End: LPWAN API
